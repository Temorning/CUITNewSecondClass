# NewSecondClass
## 进入https://ywtb.cuit.edu.cn/#/homePage，进入二课后打开开发者工具(F12)，选择网络，选择你要生成二维码的活动，在名称栏选最新的GetStuActActivityDetai，右侧请求URL标蓝部分即该二课的ID，将ID复制到程序即可。
